CC4

Computer and Network Security

Aims

  • Understanding of the principles of computer and network security.
  • Skills and competencies required to secure computer and network systems.
  • Develop secure software applications.

Learning outcomes

  • Describe the principles and practices of computer and network security.
  • Analyse and evaluate the different cryptography algorithms used in computer and network security.
  • Explain the different techniques used to exploit computer and network systems.
  • Design, implement and test security systems capable of defending computers and networks against security threats.
  • Analyse and evaluate computer and network security systems with respect to security threats.

Possible assessments

  • Test on key cryptography algorithms
  • Design, install, configure and test a secure web or cloud application server.
  • Write a report on the current threats and vulnerabilities of a computer and network system and propose a solution to defend against these threats.
  • Design, implement and test a web application that is secure against the most common vulnerabilities and attackes.
  • Final theory exam.

CC4

Advanced JavaScript

Aim

Provide the students with the theoretical knowledge and practical skills required to design, develop, and test client-side web applications.

Learning outcomes

  • Design, develop and test client-side web applications using JavaScript.
  • Understand the differences in the implementation of JavaScript by the major web browsers in current usage and to create client-side web applications that work across all browsers.
  • Evaluate the JavaScript libraries and frameworks currently available against an application’s requirements, and to utilize those libraries in developing client-side web applications.

Learning outcomes

  • Apply modern software engineering techniques, such as the Model-View-Controller architecture and asynchronous programming, in the design and implementation of client-side web applications.
  • Integrate client-side and server-side web applications using best practice programming techniques, including the use of XML, AJAX, JSON and web services.

Possible assessments

  • Design, build and test a game or similar piece interactive media using modern JS features.
  • Design, build and test a web application using a modern tech stack such as React + Express + Node + MongoDB.
  • Final theory exam.
